Model rocket control system update (version 6)

Posted on 2024-10-29 Edited on 2024-10-31 In The Control Systems Series

Lately I just re-built another iteration of my model rocket control system. I call this “v6cs”, which stands for version 6 control system. The main difference between this design and v5cs is that, unlike v5cs, which is enclosed in a separate, tubular bay, v6cs is incorporated into the nosecone section. This change aims to simplify the design (but definitly causes TONS of caos in manufacturing), make the rocket more compact and thus reduce the mass and volume of the rocket in general. In the v5cs, the nosecone has a large volume of cavity above the parachute bay left unused, contributing to unnessecary mass; v6cs perfectly addresses this issue by stuffing all of the electronics into the nosecone.

The v6cs system uses the same electronic boards and sensors, and wirings as the v5cs system. The v6cs‘s program is slightly upgraded: the servo will return to its initial position whenever the mpu6050 module senses the rocket to be upright. This correction is made because that the entire system has to be turned upside-down when plugging in the battery.
